# Frontend Visual Identity

## Design Principles

* Modern cybersecurity SaaS aesthetic
* Minimalist dark interface
* Strong contrast and readability
* Subtle neon highlights
* Consistent spacing
* Smooth and fast animations

# Component Guidelines

## Buttons

Primary buttons:

* cyan background
* white text
* subtle hover glow

Secondary buttons:

* transparent background
* subtle border
* dark hover background

## Inputs

* dark background
* subtle border
* focus ring using primary accent
* smooth transitions

## Cards

* soft shadows
* rounded-xl
* thin borders
* generous spacing

# Animation

Use Framer Motion for:

* fade transitions
* hover effects
* page transitions

Animations must be:

* fast
* subtle
* professional

Avoid exaggerated motion.

# Icons

Preferred library:

* lucide-react

Recommended icons:

* Shield
* Lock
* KeyRound
* Fingerprint
* FileSignature
* BadgeCheck

# General UX

* prioritize clarity over decoration
* avoid visual noise
* keep layouts clean and structured
* maintain accessibility and contrast
* use spacing consistently
